Notes

Wichtige Hinweise

  • Verwende Chrome oder Edge.
  • Verwende ein datenfähiges USB-Kabel.
  • Nutze einfach den passenden Installer-Button für die gewünschte Firmware.
  • Falls keine Verbindung möglich ist, Board neu anstecken und erneut versuchen.
  • Falls nötig, beim Verbinden die BOOT-Taste gedrückt halten.
  • Die Installation überschreibt die vorhandene Firmware auf dem Gerät.

Important Notes

  • Use Chrome or Edge.
  • Use a data-capable USB cable.
  • Use the installer button that matches the firmware you want.
  • If the connection fails, reconnect the board and try again.
  • If needed, keep the BOOT button pressed while connecting.
  • The installation will overwrite the existing firmware on the device.
